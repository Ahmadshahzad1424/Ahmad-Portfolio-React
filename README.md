A high-performance, visually stunning developer portfolio built with a modern React tech stack. This project showcases my journey as a **Frontend Developer** and **AI Enthusiast**, highlighting my skills, projects, and educational milestones with a premium "Apple-level" aesthetic.

## 🚀 Key Features

- **Modern UI/UX**: Premium design featuring glassmorphism, vibrant gradients, and a sleek dark theme.
- **Micro-Animations**: Sophisticated entrance animations and scroll-based effects using **Framer Motion** and **GSAP**.
- **Responsive Design**: Fully optimized for all device sizes, from mobile screens to 4K desktops.
- **Interactive Tech Stack**: A visual representation of technical fluencies with hover effects and progress indicators.
- **Dynamic Projects Showcase**: Categorized project cards with github and live preview links.
- **Contact Integration**: Fully functional contact form integrated with **Web3Forms** for seamless communication.
- **Performant Loader**: Custom-built, synchronized progress-based experience loader.

## 🛠️ Tech Stack

- **Core**: React 19 (Hooks, Functional Components)
- **Styling**: Tailwind CSS (Utility-first, Custom Design System)
- **Animations**: Framer Motion (Orchestration, Layout Animations), GSAP
- **Icons**: Lucide React, React Icons
- **Typography**: Inter (Sans), Fira Code (Mono)
- **Build Tool**: Vite (Ultra-fast development & bundling)
- **Smooth Scroll**: Lenis Scroll for refined scroll behavior

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ahmadshahzad1424/ahmad-portfolio-react.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ahmad-portfolio-react
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
